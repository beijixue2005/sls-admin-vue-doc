# time

|属性        |      说明    |    类型   |         必填     |    可选值    |        默认值 |
| --- | --- | --- | --- |----|----|
| key        |      字段名称   |  String   |    Y  |    -   |   -    |
| type        |      字段类型   |  String   |    Y  |    input   |   input    |
| label        |     label文本   |  String   |    N  |    date/daterange/year/month/week   |   -    |
| desc        |     占位符   |  String   |    N  |    -   |   -    |
| time_attrs        |      ElementUI支持的属性，看下文   |  Object   |    N  |    -   |   {}    |
| events        |      ElementUI支持的事件，看下文   |  Object   |    N  |    -   |   {}    |

### time_attrs和events参考input组件中的说明。
