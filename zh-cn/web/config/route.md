# 路由配置

### 以demo模块路由配置为例进行解释
```shell
├── src                        // 源代码
│   ├── router                 // 路由
│       ├── index.js                 // 负责初始化路由，导入此文件夹下所有目录配置的路由
│       ├──demo                 // demo模块路由
│           ├──index.js                // demo模块的头部路由配置以及导出该模块下的所有菜单路由
│           ├──article.js                // 左侧文章管理路由配置
│           ├──user.js                // 左侧用户管理路由配置
│           ├──order.js                // 左侧订单管理路由配置
```

> - 每个文件内容具体可查看代码。
> - 实现原理就是把路由数组拆分出来，最后在router/index.js中汇总，统一导入即可。
