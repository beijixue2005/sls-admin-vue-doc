## 约定

接口返回格式统一为对象，格式如下：

| 属性 | 值 | 类型 | 描述 | 
| --- | --- | --- | --- |
| status | 200->成功；1->错误；404->token无效 |Number|状态码 |
| msg | msg | String | 具体的错误信息 |
| data |{} | Object | 数据对象 |


>当 status === 200 时，data必存在，至少是一个空对象返回。

>当 status !== 200 时，msg必存在，返回具体的错误信息。

>当 status === 404 时，代表token无效，需要重新登录。
